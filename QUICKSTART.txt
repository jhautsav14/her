WHAT IS THIS?
A beautiful mobile-first interactive storytelling experience with animations,
glassmorphic UI, and immersive backgrounds. Features typewriter text effects,
floating hearts, and smooth scene transitions.

INSTANT SETUP (3 STEPS)
================================================================================

1. INSTALL DEPENDENCIES
   $ pnpm install
   (or: npm install / yarn install)

2. START DEVELOPMENT SERVER
   $ pnpm dev
   
3. OPEN IN BROWSER
   → Desktop: http://localhost:3000
   → Mobile:  http://<YOUR_IP>:3000 (find IP: ipconfig or ifconfig)

THAT'S IT! App is running ✨

================================================================================
KEY FEATURES
================================================================================

✅ 9:16 Mobile Format      | Full-screen immersive portrait layout
✅ 7 Story Scenes          | Complete narrative arc with dialogue & quotes
✅ Typewriter Animation    | Character-by-character text reveal (50ms/char)
✅ Ken Burns Zoom          | Subtle 8-second background animation
✅ Glassmorphism Card      | Semi-transparent dialogue with blur effect
✅ Progress Bar            | 7-segment tracker showing story progress
✅ Floating Hearts         | Particle effect on final scene
✅ Smooth Transitions      | Fade & slide scene changes with Framer Motion
✅ Tap Navigation          | Invisible overlay for scene progression
✅ Color Theme             | Midnight Velvet (deep indigo + soft rose)

CUSTOMIZATION (QUICK REFERENCE)
================================================================================

CHANGE STORY CONTENT
   File: /hooks/useStory.ts
   Edit: STORY_SCENES array with your dialogue and quotes

CHANGE COLORS
   File: /app/globals.css
   Edit: --midnight, --rose-accent, --white-ghost, --lavender variables

CHANGE FONTS
   File: /app/layout.tsx
   Edit: Import different fonts from Google Fonts

ADJUST TEXT SPEED
   File: /components/story/TypewriterText.tsx
   Edit: Change "50" to faster (30) or slower (100)

CHANGE BACKGROUND IMAGES
   Place images in /public/ folder
   Reference in useStory.ts: backgroundImage: '/your-image.jpg'

MORE CUSTOMIZATION? → Read QUICK_CUSTOMIZE.md

TESTING ON MOBILE
================================================================================

METHOD 1: LOCAL NETWORK (BEST)
   1. Get your IP:
      Mac:     ifconfig | grep inet
      Windows: ipconfig
   2. On phone browser:
      http://<YOUR_IP>:3000
   3. Test in portrait mode

METHOD 2: CHROME DEVTOOLS
   1. Open: http://localhost:3000
   2. Press: Ctrl+Shift+I (or Cmd+Opt+I on Mac)
   3. Click: Device toggle (⌨ at top right)

   1. Run: vercel deploy
   2. Copy deployment URL
   3. Open on phone browser

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: "Dependencies not found"
FIX:   Run: pnpm install

ISSUE: "Port 3000 already in use"
FIX:   Run: pnpm dev -- -p 3001

ISSUE: "Images not loading"
FIX:   Ensure images in /public/ folder
       Use path: /image-name.jpg (not ./image-name.jpg)

ISSUE: "Animations stuttering"
FIX:   Close other apps, clear browser cache
       Test on actual mobile device (DevTools can be slower)

ISSUE: "Mobile view not full screen"
FIX:   Clear browser cache
       Check viewport settings in layout.tsx

MORE HELP? → Read DEPLOYMENT.md or README.md

================================================================================
TIPS FOR BEST RESULTS
================================================================================

✓ Test on actual mobile device (not just DevTools)
✓ Keep scenes between 5-12 for good pacing
✓ Use 9:16 aspect ratio images (1080x1920px)
✓ Keep dialogue under 100 words per scene
✓ Use high-contrast colors for readability
✓ Deploy to Vercel for instant CDN + analytics

✗ Don't add more than 20 scenes
✗ Don't use very long dialogue blocks
✗ Don't skip testing on real mobile device
✗ Don't forget the "/" in image paths (/image.jpg)
